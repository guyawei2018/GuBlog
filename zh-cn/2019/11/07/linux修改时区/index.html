Linux修改时区

作者： GUYW / 2019-11-07 / 分类： Notes

notes

查看系统时区

    [root@linux-node ~]# date -R

修改系统时间

    1、将日期设置为2017年11月3日
    [root@linux-node ~]# date -s 11/03/17

    2、将时间设置为14点20分50秒
    [root@linux-node ~]# date -s 14:20:50

    3、将时间设置为2017年11月3日14点16分30秒（MMDDhhmmYYYY.ss）
    [root@linux-node ~]# date 1103141617.30

时区设置用tzselect 命令来实现

    但是通过tzselect命令设置TZ这个环境变量来选择的时区，需要将变量添加到.profile文件中

    执行tzselect命令 --> 选择Asia --> 选择China --> 选择east China - Beijing, Guangdong, Shanghai, etc-->然后输入1。

    执行完tzselect命令选择时区后，时区并没有更改，只是在命令最后提示你可以执行 
    TZ=’Asia/Shanghai’; export TZ 并将这行命令添加到.profile中，然后退出并重新登录。
    这里可以将TZ=’Asia/Shanghai’; export TZ添加到/etc/profile文件中，最后source profile文件
